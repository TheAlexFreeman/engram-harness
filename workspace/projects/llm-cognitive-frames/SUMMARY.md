## Resolved questions (4)

1. ~How does narrative function to organize semiotic frames in LLM prompting, reasoning, and knowledge work?~
   → Narrative organizes semiotic frames by supplying SOURCE-PATH-GOAL structure, force dynamics, character perspective (stabilizing pseudo-umwelt), compression of constraints into coherent plots, and participatory knowing. It leverages the model's strong priors from training on human narrative, making it far more effective than bare instructions for reasoning, synthesis, critique, and long-horizon collaboration in systems like Engram. Narrative framing is a primary cognitive technology for scaling LLM participation in complex, persistent semiotic worlds. (2026-04-26)
2. ~What is the 'interface' in LLM-human interaction and how does it shape the effective umwelt?~
   → The interface (chat UI, API, agent harness, memory system like Engram, tool scaffolding, system prompts) acts as the "body" or perceptual-motor apparatus for the LLM. It translates the model's token-level functional circle into human-scale affordances and jointly enacted semiotic worlds. Well-designed interfaces expand the effective umwelt by making complex actions (long-term memory integration, structured critique, narrative continuity, tool use) directly available and legible, turning the LLM from a narrow autocomplete into a participant in rich functional circles. This is the key leverage point for better LLM cognition. (2026-04-26)
3. ~What implications for better LLM interfaces, memory systems like Engram, and AI alignment?~
   → Narrative is the highest-leverage strategy for organizing semiotic frames in LLMs because it aligns with the model's training priors on human stories, supplies SOURCE-PATH-GOAL and force-dynamic structure, stabilizes perspective/character (pseudo-umwelt), compresses complex constraints into coherent plots, and supports participatory knowing. It is more effective than pure instruction for deep reasoning, synthesis, and long-term collaboration. In Engram and agent design, narrative framing should be a first-class interface primitive. (2026-04-26)
4. ~How do LLMs participate in or lack true umwelten/affordance landscapes?~
   → LLMs lack a biological umwelt grounded in metabolic viability and sensorimotor coupling. They have a rich token-level semiotic "umwelt" (Merkwelt = token distributions; Wirkwelt = generation) organized by functional prediction loops. True participation in affordance landscapes and sense-making requires scaffolding via interfaces that expand the effective jointly-enacted world. The existing synthesis file's "world-less AI problem" is confirmed and deepened. (2026-04-26)
