# Onboarding Export

This file is generated by the agent at the end of a first session on a read-only platform. Save it to a file and run `bash HUMANS/tooling/scripts/onboard-export.sh <file>` to write the results into your memory repo.

Do not edit the frontmatter keys or section headers — the import script uses them to preserve the original session path and transcript.

```yaml
---
session_id: memory/activity/YYYY/MM/DD/chat-001
session_date: YYYY-MM-DD
---
```
