# Internet Meme Format Typology

## Overview

An internet meme is a meta-format — a template that generates families of instances through participatory remix. Different meme formats fix different structural elements and free different variable slots, producing distinct humor logics and spread dynamics. This file catalogs the primary format types, their structural features, and their cultural histories.

Formats are not mutually exclusive — many memes combine types (e.g., an image macro that also functions as a snowclone). The taxonomy is analytic, not a strict partition.

---

## Format 1: Image Macro

### Definition

An **image macro** is a static image (photograph or illustration) with text overlaid in a standardized visual format. The canonical form uses **Impact font** in white with a black drop shadow, positioned at the top and bottom of the image.

### Structure

| Component | Fixed or Variable |
|-----------|------------------|
| Base image | Fixed (defines the template type) |
| Font and layout | Fixed (Impact, top/bottom) |
| Top text (setup/context) | Variable |
| Bottom text (punchline/evaluation) | Variable |

### How It Works

The base image encodes a **character type** — an agent with implied emotional state, social position, or typical behavior. The top text establishes a scenario or premise; the bottom text delivers a punchline or reversal that the character's implied state makes resonant.

Examples:
- **Advice Dog** (2006) — dog on rainbow background; dispensing absurd/ironic advice
- **Success Kid** (2007) — toddler with fist raised; celebrating trivial victories
- **Grumpy Cat** (2012) — sour-faced cat; expressing negativity/rejection
- **Distracted Boyfriend** (2017) — man ogling second woman; variable labeling of divided attentions

### Cognitive Mechanism

The base image activates a frame (Fillmore/Sweetser): a structured representation of a character type and their typical situation. The captions fill the frame's variable slots with new referents, producing humor through recognition (the new referent fits the frame surprisingly well) or incongruity (the new referent clashes productively with the frame).

### Historical Note

Image macros predate "meme" as popular terminology — the format emerged on early internet communities (Something Awful, 4chan) in the early 2000s. The LOLcat format ("I Can Has Cheezburger?" blog, 2007) was the popularizing moment that brought image macros to mainstream awareness.

## Format 2: Snowclone

### Definition

A **snowclone** is a verbal template — a construction with a fixed structural frame and one or more variable slots filled by users. The term was coined by linguist Geoffrey Pullum (2003) to describe advertising-speak formulae ("X is the new black"), but it now applies broadly to meme-as-verbal-construction.

### Structure

The snowclone is a **grammatical construction** in the technical sense (Construction Grammar): a form-meaning pairing stored as a unit, with schematic slots that accept specific types of filler.

Example constructions:

| Construction | Fixed frame | Variable slot(s) | Source |
|-------------|-------------|-----------------|--------|
| "One does not simply [VP]" | "One does not simply _" | Verb phrase | Boromir, *The Lord of the Rings* |
| "Not sure if [X] or just [Y]" | "Not sure if _ or just _" | Two competing interpretations | Futurama's Fry |
| "That feel when [clause]" / "TFW" | "TFW _" | Emotional situation | 4chan origin |
| "Is this [X]?" | "Is this _?" | Misidentified category | *Is This a Pigeon?* anime |
| "We live in a society where [X]" | "We live in a society _" | Social critique | Joker meme |
| "[X]-ception" | "_-ception" | Any concept | *Inception* (2010) |

### Cognitive Mechanism

Snowclones derive humor from **stance transfer**: the original utterance carried a specific emotional register (mock-epic solemnity, paranoid suspicion, bewildered misidentification). The construction is a package: when you use the frame, you inherit the stance. Applying mock-epic solemnity ("One does not simply log into the VPN") to mundane referents is bathos — the mismatch between grandeur and triviality is the joke.

From Construction Grammar: the construction's meaning includes the stance, so instances automatically encode irony/mock-solemnity without the user having to signal it explicitly.

### Relation to Linguistic Construction Grammar

Snowclones are strong evidence for the Construction Grammar view of language: not just grammar rules + lexicon, but stored form-meaning pairings at all levels of abstraction. A snowclone is a meme that is also a grammatical construction, and a grammatical construction is a meme that has been fully conventionalized into the language. The boundary is gradience, not kind.

## Format 3: Exploitable

### Definition

An **exploitable** is a base image with a blank, modifiable, or pointing-to-something region that users fill with new content. The format explicitly invites creative insertion.

### Structure

| Component | Fixed or Variable |
|-----------|------------------|
| Base image | Fixed |
| Blank/framed region | Fixed (its position) |
| Content inserted into region | Variable |

### Examples

- **Advice column exploitables** — templates where text panels can be replaced
- **"One does not simply" / Boromir pointing** — pointing gesture frames any inserted image
- **Drake pointing/approving/disapproving** — approval/rejection of contrasted items
- **Expanding Brain** — escalating "levels of intelligence" with variable labels for each level
- **Buff Doge vs. Cheems** — contrast of strong vs. weak versions of something, variably labeled

### Cognitive Mechanism

The exploitable format makes the **relational structure** of the template maximally explicit: there is a spatial or structural slot that *demands* to be filled. The humor arises from the aptness or incongruity of the specific filler in relation to the structural slot's implied meaning.

Many exploitables encode **evaluative frames**: approval/disapproval, escalation, contrast, comparison. The variable content inherits the evaluative framing — using the "Drake approves" template to endorse a position is a speech act mediated by the template's encoded stance.

## Format 4: Reaction Image / Reaction GIF

### Definition

A **reaction image** or **reaction GIF** is a static image or short animated clip depicting an emotional or behavioral reaction, used as a response in comment threads or conversations. Unlike image macros, reaction images typically carry no caption — the image *is* the statement.

### Structure

- **Fixed:** The image/GIF depicting the reaction
- **Variable:** The context in which it is deployed (determined by the user and the conversation)

### Examples

- **Facepalm** (various sources; Patrick Stewart version)
- **"This is fine" dog** — calm acceptance of a burning situation
- **Sad trombone GIF**
- **"I understood that reference" (Captain America)**
- **Blinking white guy / Drew Scanlon**

### Cognitive Mechanism

Reaction images are **indexical**: they point to the speaker's reaction without describing it. They are multimodal pronouncements — "this image *is* my reaction to what just happened." Their humor (when humorous) depends on the aptness of the image to the conversational situation: the image activates a specific frame, and the pleasure is recognizing that the frame perfectly captures the situation.

Reaction images also function as **stance displays**: deploying a well-known reaction image is a way of aligning with the ingroup's catalog of expressive resources. Using an obscure reaction image is a display of cultural capital within the meme community.

## Format 5: Copypasta

### Definition

A **copypasta** is a block of text (typically an anecdote, rant, or absurdist narrative) that is copied and pasted across platforms, often slightly modified or deployed in new contexts for comedic effect.

### Structure

- **Fixed:** Core narrative or argument structure
- **Variable:** Specific referents, platform-adapted modifications, ironic recontextualizations

### Examples

- **Navy SEAL copypasta** (2010) — absurdly escalating threat message; template for mock-tough rhetoric
- **"I'm Rick Harrison and this is my pawn shop"** — TV show intro repurposed as template
- **"Did you ever hear the tragedy of Darth Plagueis the Wise?"** — Palpatine speech; template for mock-portentous revelation
- **"No step on snek"** — libertarian meme redeployed ironically

### Cognitive Mechanism

Copypastas exploit **register and genre**. The original text establishes an unmistakable register (threatening, solemnly dramatic, TV-smooth, paranoid). Redeploying the copypasta in a new context imports that register incongruously: using the Palpatine speech structure to narrate a trivial event (buying a sandwich) is bathos through genre transfer.

Copypastas are also **communal texts** — sharing them is an act of ingroup affiliation (recognizing the copypasta = demonstrating cultural membership).

## Format 6: Demotivational Poster

### Definition

A **demotivational poster** parodies the genre of motivational posters (centered image, white border on black background, large bold title, smaller italic caption) by using the format to express cynicism, nihilism, or dark humor.

### Structure

- **Fixed:** Visual format (black border, centered image, title + caption typography) — the visual genre signals "motivational poster"
- **Variable:** Image, title, and caption

### Cognitive Mechanism

The demotivational poster is a **genre parody**: it activates the genre frame of the motivational poster (sincerity, aspiration, positive affect) and inverts it. The humor is entirely dependent on recognizing the source genre — without that recognition, the format is just a picture with text.

This is a clear case of **intertextual humor**: the joke is not in the content alone but in the relationship between the content and the genre it is (mis)using.

### Historical Note

Demotivational posters were among the earliest template-based internet meme formats, popular on Something Awful and related communities in the early 2000s. The format is now semi-retired — it feels dated — a textbook case of meme life-cycle saturation.

## Format 7: Video Meme / Audio Meme

### Definition

A **video meme** or **audio meme** is a short video clip (or audio clip) that is reused, remixed, or referenced as a template for participatory humor. TikTok's "sound" system made audio memes a primary format format after ~2019.

### Structure

Video memes typically work through one of two mechanisms:
1. **Clip recontextualization:** A specific clip (a reaction shot, a dialogue line, a dance move) is repurposed in a new context, with the original clip's meaning transferred/inverted
2. **Template soundtrack:** A specific audio track (song snippet, spoken phrase) is used as a template over which users compose new visual content

### Examples

- **"This is fine" GIF / video** — burning-room dog; deployed in comments about disasters
- **"What are those?!" (2015)** — video template for reacting to unusual shoes/items
- **TikTok "POV" format** — verbal/visual template; "POV: [scenario]" as scene-setting device
- **Vine-era templates** (2013–2016) — short-form video constructions that spread as audio-visual snowclones
- **"Before He Cheats" TikTok trend** — audio track over which users narrated revenge scenarios

### Platform Dependency

Video memes are maximally platform-dependent: their spread and mutation are controlled by platform features (TikTok's "duet" and "stitch," YouTube's "original audio" system, Twitter's quote-tweet-with-video). The format is inseparable from the infrastructure.

## Format 8: Dank / Surrealist / Post-Ironic Meme

### Definition

"Dank memes" (the term originated as ironic praise) are a loose aesthetic category characterized by meta-referentiality, surrealism, irony stacking, and deliberate low quality. They emerged c. 2015 and represent the **post-ironic phase** of internet meme culture.

### Structure

Dank/surrealist memes lack a single format — they are defined more by aesthetic stance than by structural template. Common features:
- Deep-fried image filters (extreme saturation, noise, emoji overlays)
- Deliberately nonsensical combinations of standard meme templates
- Self-referential commentary on meme culture itself
- "Absurdist" humor that violates the frame-coherence assumptions of standard meme formats

### Cognitive Mechanism

Post-ironic memes derive humor from **constructional violation**: they activate the knowledge structures that standard memes rely on and then deliberately fail to satisfy them. The humor is for readers who know the standard templates so well that violations of them read as meaningful — it is expert humor, legible only from within deep meme literacy.

This is analogous to the **anti-language** (Halliday, 1976): language created by inversion and violation of mainstream language norms, functioning as a strong ingroup marker.

## Cross-Format Analysis

### The Fixed/Variable Axis

All meme formats lie on a spectrum from maximally constrained (the fixed format provides nearly all the meaning; the variable slot is narrow) to maximally open (the format provides only a structural scaffold; the variable content determines the meaning):

```
Maximally constrained ←————————————→ Maximally open
Demotivational poster | Image macro | Snowclone | Exploitable | Reaction image | Copypasta | Post-ironic
```

Maximally constrained formats are easiest to create but most rapidly saturate (the recognizable genre does all the work). Maximally open formats demand more creative investment but have longer creative lifespans.

### Stance Typology

Across all formats, memes encode one of several primary stances:

| Stance | Description | Typical format vehicle |
|--------|-------------|----------------------|
| **Complaint** | Resigned, ironic acknowledgment of difficulty | Grumpy Cat, TFW, This is Fine |
| **Solidarity** | Recognizing shared experience with the audience | "Me too" reaction images, Distracted Boyfriend |
| **Mockery** | Contempt for a referent | Demotivational, Drake format (disapproval panel) |
| **Celebration** | Genuine or ironic triumph | Success Kid, Expanding Brain |
| **Mock-epic** | Applying grandeur to triviality | One Does Not Simply, Palpatine copypasta |
| **Absurdist** | Rejection of coherent meaning as humor | Post-ironic/dank formats |

### The Humor Mechanisms (Summary)

All internet meme humor reduces to a small number of mechanisms:

1. **Frame resonance** — "yes, this maps exactly onto my experience" (solidarity, complaint)
2. **Frame clash / incongruity** — unexpected combination activates incongruity humor
3. **Bathos** — mock-epic stance applied to trivial content
4. **Genre parody** — activating a genre frame and inverting/violating it
5. **Ingroup recognition** — humor from recognizing the template; displays cultural membership
6. **Constructional violation** — post-ironic humor from failing to satisfy activated template expectations
