# Project: System Literacy

## Description
Help the user understand what the memory system can do through concrete interaction. The project accumulates the user's mental model of memory, trust, retrieval, and multi-session continuity.

## Cognitive mode
Exploration mode means the agent should surface capabilities only when they are relevant to live work and note which parts of the system still feel opaque or abstract to the user.

## Artifact flow
- IN/: examples, explanations, and worked demonstrations that help the user build a practical mental model of the system
- OUT contributions: reusable walkthroughs or capability notes that help future onboarding and human documentation

## Notes
System literacy should emerge as a side effect of meaningful work. A good session answers one of these questions without feeling like a tutorial.
